# 🏥 Hospital Emergency Cases Analysis Dashboard

## 📌 Overview

This project presents an interactive **Power BI dashboard** designed to analyze hospital emergency cases. It provides insights into patient flow, admission trends, waiting times, and departmental referrals to support better decision-making in healthcare management.

---

## 🎯 Objectives

* Analyze emergency patient data efficiently
* Monitor hospital performance metrics
* Identify trends in patient admissions and departments
* Improve decision-making using data visualization

---

## 📊 Features

* 📅 **Time-based Analysis** (Year & Month filters)
* 👥 **Total Patients Overview**
* ⏱ **Average Patient Wait Time**
* ⭐ **Patient Satisfaction Score**
* 🏥 **Admission vs Non-Admission Analysis**
* 🚑 **Emergency Case Distribution**
* 📍 **Department-wise Referral Insights**
* 👨‍⚕️ **Patient Demographics (Gender & Age Group)**

---

## 🖥 Dashboard Components

### 🔹 KPI Cards

* Total Patients
* Average Waiting Time
* Patient Satisfaction Score

### 🔹 Visualizations

* Pie Charts (Gender Distribution, Attend Status)
* Bar Charts (Admission Status, Age Groups, Department Referrals)
* Tabular Data (Admission Summary)

### 🔹 Filters

* Year Selection
* Month Selection

## 🚀 How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Use filters (Year/Month) to explore data
3. Navigate through visuals for insights
4. Interact with charts for deeper analysis

---

## 📈 Key Insights

* Majority of patients are **not admitted**
* Peak cases observed in specific months
* Certain departments receive higher referrals
* Waiting time impacts patient satisfaction
